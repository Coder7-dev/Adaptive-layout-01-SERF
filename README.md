Название: Лендинг Surfing School

Стек: HTML, CSS (Flex, Grid)

Задачи:
1. Разработал адаптивную верстку по подходу Desktop-First. 
2. Настроил валидную, семантическую верстку для улучшения SEO-показателей сайта.

Фичи: 
1. Реализовал скрытый h1 заголовок (Surfing school) с помощью класса visually-hidden. Это позволило сохранить правильную семантическую структуру документа для поисковых роботов (SEO) и скринридеров, не нарушая дизайн-макет.
2. Реализовал сетку интерактивных круглых карточек с использованием современного CSS-свойства aspect-ratio, обеспечив сохранение идеальных пропорций элементов при любых изменениях ширины экрана.
3. Реализовал волнообразный визуальный сдвиг карточек интерфейса без засорения HTML-разметки, применив псевдокласс :nth-child(even).
